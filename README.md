# Inventory Management CRUD App

A responsive Inventory Management CRUD application built with Next.js, Drizzle ORM, Neon PostgreSQL, Cloudinary, and Tailwind CSS.

## Features

- Create inventory items
- Read inventory items
- Update inventory items
- Delete inventory items
- Upload and store images using Cloudinary
- Store product data in Neon PostgreSQL
- Responsive Tailwind CSS UI

## Tech Stack

- Next.js
- Tailwind CSS
- Drizzle ORM
- Neon PostgreSQL
- Cloudinary

## Run Project

```bash
npm install
npm run dev
